# Loyalty Engine

## About Loyalty Engine

Loyalty Engine is an API-first loyalty management platform with a dashboard UI. The project is currently not deployed. It features:
- **Coupons** - Granting and management
- **Vouchers** - Voucher creation
- **Redemptions** - Coupons, vouchers redemptions
- **Wallets** - Wallet management
- **Points** - Points granting and exchange/redeem
- **Rankings** - Customer rankings and leaderboards
- **Badges** - Achievement badges and rewards
- And more
